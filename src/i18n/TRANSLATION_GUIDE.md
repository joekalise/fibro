# Fibro — Translation Style Guide & Glossary

Reference for translators. Covers tone, voice rules, and term-by-term guidance for Spanish (Spain) and Spanish (United States / Latin American).

---

## Style Guide

### Fixed English terms (never translate)
These appear in Spanish copy as-is:
- **Fibro** — the app name, always capitalised, always English
- **Fibro Score** — branded score name
- **Premium** — subscription tier
- **HRV** — abbreviation kept in all locales (spell out once on first use if needed: "HRV (variabilidad de la frecuencia cardíaca)")
- **FIQ / FIQ-R** — clinical instrument name, kept in English

### Tone & voice
- Warm, direct, like a knowledgeable friend who also has fibromyalgia
- Never clinical, never alarming
- Use real numbers and data when talking about patterns ("tu dolor promedio fue 7/10" not "tu dolor ha sido alto")
- Don't soften bad news into vague encouragement
- Never start a sentence with a diagnosis-sounding phrase

### Form of address
| | es-ES | es-US |
|---|---|---|
| Singular | **tú** (informal) | **tú** (informal) |
| Plural | Avoid vosotros — rephrase to singular or use "las personas" | No vosotros — use ustedes if plural is needed |
| Imperative | "Registra", "Guarda", "Toca" | Same |

### Punctuation & formatting
- Inverted question/exclamation marks: **¿…?** and **¡…!** in both locales
- Decimal separator: comma in es-ES ("7,2/10"), period in es-US ("7.2/10")
- Date format: es-ES uses DD/MM/AAAA; es-US uses MM/DD/AAAA (but the app uses relative labels like "Hoy", "Ayer" — these are fine as-is)

## Things to watch

- **"Brote" vs "crisis"** is the single most important regional split. Use it consistently within a locale — don't mix them.
- **Gender agreement**: when referring to the user generically, prefer neutral constructions ("tu nivel de dolor") over gendered ones. Where unavoidable, use the slash form ("descansado/a") for now — a later pass can use detected biological sex.
- **"Fibro"** always stays exactly as written — never "la Fibro" or "el Fibro". Treat it as a proper noun with no article.
- **Medical terms** (niebla mental, rigidez matutina, sueño no reparador) should match what people with fibromyalgia in each region actually use online and in patient communities — don't default to overly clinical dictionary translations.
